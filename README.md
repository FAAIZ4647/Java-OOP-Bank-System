# 🏦 Java OOP Bank System

A simple **Java command-line banking application** that demonstrates **Object-Oriented Programming (OOP)** concepts such as abstraction, inheritance, and method overriding.

The system allows a user to:

* Deposit money (with interest added)
* Withdraw money (with service charge applied)
* Check account balance

---

# 📌 Project Overview

This project simulates a basic **bank account management system** implemented using Java.

It uses an **abstract class (`Account`)** that defines common banking operations, and a subclass **(`CustomerAccount`)** that provides concrete implementations for deposit and withdrawal operations.

The application interacts with the user through the **command line**.

---

# ⚙️ Features

* Deposit money with **2% interest**
* Withdraw money with **1% service charge**
* Display available balance
* Input handling using `Scanner`
* Implementation of **abstract classes**
* Demonstrates **inheritance and method overriding**

---

# 🧠 Concepts Used

This project demonstrates key Java concepts:

* Abstract Classes

* Encapsulation
* User input using `Scanner`

# ▶️ How to Run the Program

### 1️⃣ Clone the repository

```
git clone https://github.com/FAAIZ4647/Java-OOP-Bank-System.git
```

### 2️⃣ Navigate to project folder

```
cd Java-OOP-Bank-System
```

### 3️⃣ Compile the Java file

```
javac BankSystem.java
```

### 4️⃣ Run the program

```
java BankSystem
```
